## Ejecutable JAR

Se ha compilado el proyecto y generado un archivo `Biblioteca.jar` en el directorio raíz. Para ejecutar en cualquier equipo con Java instalado:

```bash
java -jar Biblioteca.jar
```

El jar contiene todas las clases necesarias y usa `Main` como clase principal.
